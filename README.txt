Not sure why my chrome extension stopped working so decided to create something own to download videos from youtube.

Concept: 'http://www.youtube.com/get_video_info?video_id=' . $_GET['id']
Secret: &el=detailpage - avoids copyrighted video restrictions

Usage: http://centos/youtube-downloader-php/?id=FszEaxrHGTs

you probably need to rename or add file extension if it's not mp4 ...
